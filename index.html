About this Project

Welcome! These pages use data to answer questions about weather. Specifically, how do different weather elements - cloudiness, humidity, temperature, and wind speed - vary by latitude?

The starting data was generated using random latitude and longitude to find 500+ cities around the world (find the full data set in the data page). The visualization pages all show scatter plots to identify correlation, if any, between latitude and the given weather element.
